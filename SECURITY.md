# Security Policy

## Scope

This repository is proprietary software developed for the internal use
of SMB CPA Group, PC. It is not an open-source project and no public
support is offered.

## Reporting

- Report security issues privately through the firm's secure internal
  channel.
- Do not file public issues or pull requests for security reports.
- No public bug bounty or public disclosure intake is offered for this
  repository.

## CI / security workflow

No public CI or security reporting workflow is configured for this
repository by design. Internal validation and acceptance follow the
sign-off gate process documented in `CLAUDE.md` and per-category
`_SIGNOFF.md` companions under `__strategy_library/subcategories/`.
